# 🧠 AlphaMind AI

## Financial Market Intelligence Platform

AlphaMind AI is an AI-powered financial analytics platform that combines real-time stock market data with Natural Language Processing (NLP) based news sentiment analysis to generate market intelligence insights.

The platform analyzes selected companies, visualizes stock trends, evaluates financial news sentiment, and provides an overall market sentiment summary through an interactive dashboard.

---

## 🚀 Features

### 📈 Real-Time Stock Analysis
- Fetches live stock market information
- Displays:
  - Current stock price
  - Market capitalization
  - P/E ratio
- Visualizes historical stock movement

### 📰 Financial News Intelligence
- Collects recent financial news related to selected companies
- Processes news articles using NLP techniques
- Classifies market sentiment:
  - Positive
  - Negative
  - Neutral

### 🤖 AI-Based Sentiment Analysis
- Uses Natural Language Processing to analyze financial articles
- Converts unstructured news data into actionable market insights

### 📊 Interactive Dashboard
- Built using Streamlit
- Interactive stock trend visualization using Plotly
- Company-wise analysis interface
